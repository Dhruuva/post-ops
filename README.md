# post-ops

A boilerplate that implements Google, Mail.ru, Github, Facebook and classic Authentication. So it can be easy to implement it as a starting point for your projects. In the root folder You will find the frontend classic vue (+vuetify) project. This Client consists of two areas for authenticated users and guest users.
